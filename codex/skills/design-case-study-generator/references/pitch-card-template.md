# Pitch card template (3 options)

Use this exact format for each option.

## Option <A|B|C>: <name>

**Vibe**: <3–6 words>

**Sell**: <1–2 sentences. Make it feel desirable.>

- **Best for**: <who + when>
- **Core loop**: <1 sentence describing the interaction loop>
- **Layout rhythm**: <dense vs airy; grid vs freeform; hierarchy cues>
- **Type + voice**: <type feel + content tone>
- **Color + contrast**: <high contrast vs muted; accent strategy>
- **Shape language**: <radii, borders, shadows>
- **Motion**: <snappy vs calm; what animates>
- **UX signatures**: <1–3 signature patterns>
- **Risks**: <1–3 risks>
- **Demo will include**:
  - <screen/state 1>
  - <screen/state 2>
  - <screen/state 3>
